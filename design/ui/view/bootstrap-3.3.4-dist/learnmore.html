Project Platform

自身のスキルを磨く個人様、プロジェクトのメンバを招集したいプロジェクトマネージャ様へプロジェクトとメンバ探しをお手伝いします。

　

ルールは簡単。

プロジェクトマネージャ様、個人様それぞれがアカウントをご登録すること。

プロジェクトマネージャ様、個人様それぞれが情報を公開すること。

同一のプロジェクトまたは個人様に対して発信するスカウトや参加希望はひと月に１回。

　

以上のルールからコミュニケーションのきっかけをおつくり致します。

　

注意！

スカウトまたは参加希望を行った場合、相手に自身が登録したメールアドレスを提供することになります。
